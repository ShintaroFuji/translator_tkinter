# 概要  
Tkinterを使用したシンプルな翻訳アプリケーション。Google Translate APIを利用して、複数の言語間での翻訳をサポートしています。

# 機能  
複数の言語に対応
簡単なGUIで直感的な操作
テキストの保存機能

# 使用方法  
アプリを起動します。
左のボックスに翻訳したいテキストを入力します。
ソース言語とターゲット言語を選択します。
「翻訳」ボタンをクリックして、右のボックスに翻訳されたテキストを表示します。
